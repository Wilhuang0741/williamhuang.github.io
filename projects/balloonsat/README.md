# Bronco Star BalloonSat Power System

## Overview

Served as Power Systems Subteam Lead for a high-altitude BalloonSat project operating near 100,000 feet.

## Responsibilities

- Led electrical power system design and integration
- Designed and validated a 7.4V Li-ion power system
- Developed voltage regulation and protection circuitry
- Integrated sensors and embedded hardware
- Performed system-level troubleshooting using oscilloscopes and DMMs
- Supported testing and deployment of the BalloonSat platform

## Technologies

- Power Systems
- Voltage Regulation
- Embedded Hardware
- System Integration
- Oscilloscopes
- Digital Multimeters (DMM)

## Outcome

Successfully completed the BalloonSat project during Spring 2026.
